Создание JavaScript

В 90-е появилась необходимость делать веб-страницы более «живыми» и динамичными: создавать анимации, взаимодействие с пользователями и другие виды интерактивности. Нужен был легкий скриптовый язык, но не предназначенный для крупных разработчиков и людей, имевших отношение к инженерной стороне вопроса (эту нишу в то время уже заняла Java).

Эти идеи Марка Андриссена (основателя Netscape Communications) поддержал Брендан Айк, которого считают отцом JavaScript. Прототип нужно было подготовить в кратчайшие сроки. Изначально планировалось создать аналог Scheme (мощный диалект Lisp с максимально упрощенным синтаксисом) для браузера, но после команда стала разрабатывать нечто иное, похожее по синтаксису на Java. Так в 1995 году появился прототип Mocha, позднее переименованный в LiveScript, а после и в JavaScript.

Особенности JavaScript

Что отличает JavaScript от других языков программирования

Легкость запуска

Скрипты выполняются без компилляции или специальной подготовки, поэтому JavaScript может выполняться на сервере, в браузере и на любом устройстве, на котором есть "движок JavaScript"

Безопасность

Возможности языка в браузере ограничены, чтобы предотвратить доступ веб-страницы к личной информации. JavaScript на веб-страницы не имеют прямого доступа к системным функциям ОС, разные вкладки и окна "не знают" друг о друге

Удобство

JavaScript полностью интегрирован с HTML и CSS и поддерживается всеми основными браузерами. Его удобно использовать для написания небольшого, несложного кода, что делает его очень удобным для веб-разработки

Веб-разработка

JavaScript часто применяется вместе с HTML и CSS для разработки веб-страниц и не только

HTML

HyperText Markup Language («язык гипертекстовой разметки») определяет содержание и структуру веб-контента.

  • Для создания элементов используются теги
  • Множество семантических тегов
  • Нет привязки к средству воспроизведения
Документация

CSS

Cascading Style Sheets («каскадные таблицы стилей») используются для представления внешнего вида документа, написанного на HTML или XML.

  • Правила в самом документе или отдельном файле
  • Описания типа "свойство: значение"
  • Селекторы для применения правил
Документация

JavaScript

JavaScript это прототипно-ориентированный, мультипарадигменный язык с динамической типизацией.

  • Полная интеграция с HTML и CSS
  • Поддержка всеми основными браузерами
  • Скриптам не нужна подготовка для запуска

IDE для JavaScript

Лучшие редакторы кода для работы с JavaScript

WebStorm

Интегрированная среда разработки для JavaScript и связанных технологий от JetBrains. Есть бесплатный пробный период на 30 дней.

Visual Studio Code

Бесплатная среда разработки, которая поддерживает работу с многими языками программирования и технологиями.

Brackets

Легкая, быстрая и бесплатная среда разработки от Adobe с множеством настроек. Создана специально для фронтенд-разработки.

Бесплатная среда разработки для нескольких языков программирования. Есть специальная версия редактора для работы с JavaScript.
